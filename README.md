##	Get Started

In CLI:
```bash
# Global command "clisp" will be installed.
npm install -g clisp

# Show help info.
clisp ?

# Show available operators.
clisp @

# Run.
clisp + 1 2 3
# OUTPUT: 6
```

In Node.js: 
```javascript
const clisp = require('clisp');

clisp.plus(1, 2); 
// RETURN 3
```
